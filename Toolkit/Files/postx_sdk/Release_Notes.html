1. About this release

IronPort software release version numbers consist of three numeric values, separated by periods, as in major.minor.update. A change in the major.minor revision level indicates significant improvements in functionality. Updates generally consist of bug fixes to the base code at the same release level and compatible extensions to existing functionality.

From time to time, IronPort may skip one or more numbers. This may reflect a release that was distributed to a limited number of customers or a number that was reserved for another purpose but never used. Releases and updates are installed only at the customer's request.

About the IronPort SDK 6.4 release

IronPort SDK 6.4 is a generally available release. It supersedes IronPort SDK 6.3 and all previous releases. The IronPort SDK 6.4 release notes are included below for convenience.

2. Upgrade notes

PostX SDK 6.2.8h has new configurations and the daemon uses new shared libraries. Be sure to replace the entire PostX configuration file and libraries when updating the PostX SDK daemon.
